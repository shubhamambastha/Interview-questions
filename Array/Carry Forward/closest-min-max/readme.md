## Question:

Given an array A, find the size of the smallest subarray such that it contains at least one occurrence of the maximum value of the array and at least one occurrence of the minimum value of the array.

**Problem Constraints:**

```
1 <= |A| <= 2000
```

---

### Example:

```
Example 1:-
A = [1, 3];
ans = 2;

Example 2:-
A = [2];
ans = 1;
```
