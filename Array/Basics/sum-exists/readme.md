## Question:

Given N array elements, Check if there exists a pair (i, j) such that `arr[i] + arr[j] === K` && `i !== j`.
Note: i and j are index values, k is given sum.

---

### Example:

```
Example 1:-
arr[7] = [ 3, -2, 1, 4, 3, 6, 8 ]; k = 10
ans = true;

Example 2:-
arr[4] = [ 2, 4, -3, 7 ]; k = 5
ans = false;
```
