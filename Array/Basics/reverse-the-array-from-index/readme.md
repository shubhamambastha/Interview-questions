## Question:

Given an array and a range `[si, ei]`. Reverse the array within `[si, ei]`
Note: `si <= ei`

**Problem Constraints:**

```
1 <= A.size() <= 10000
1 <= A[i] <= 10000
1 <= si, ei <= A.size()
si <= ei
```

---

### Example:

```
Example 1:-
arr[9] = [ -3, 4, 2, 2, 6, 9, 7, 8 ,10 ];
si = 3; ei = 7
ans = [ -3, 4, 2, 8, 7, 9, 6, 2, 10 ];
```
